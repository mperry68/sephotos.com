# sephoto.com

Photography studio website with English and French language support.

## Folder Structure

- `/public` - Static assets served directly
- `/public/images` - Photography images and galleries
- `/public/assets` - Other assets (fonts, icons, etc.)
  - `/public/assets/fonts` - Custom font files
  - `/public/assets/css` - Stylesheets including `fonts.css`
- `/content` - Language-specific content files
- `/locales` - Translation files (if using i18n library)

## Languages

- English (default): `/content/en/`
- French: `/content/fr/`

## Deployment

This site is configured for Cloudflare Pages deployment.
